À qui s'adresse ce guide ?

Ce guide suit un scénario concret : celui d'un bailleur qui possède plusieurs logements (Appartement A, B, C) et qui souhaite piloter à distance l'électricité de chacun depuis un seul téléphone — de l'installation jusqu'à la gestion fine des lignes, et le partage de configuration entre deux téléphones par QR Code.

Le principe d'EnMKit. Chaque logement est équipé d'un kit (un boîtier relié à une carte SIM). L'application envoie des SMS de commande au numéro du kit pour allumer / couper les circuits (lignes) et relever la consommation. Aucune connexion Internet n'est requise : tout passe par le réseau GSM. Quand le kit confirme, tous les téléphones autorisés se synchronisent — même application fermée — et chaque confirmation est conservée dans un historique horodaté par ligne (§6.3).

Code d'accès — proposé, mais skippable

Nouveau Une sécurité proposée, jamais imposée

Juste après l'introduction, l'app propose de créer un code — mais un bouton « Passer pour l'instant » permet d'aller directement au tableau de bord.

  • Créer un code (4 chiffres) → protège l'ouverture de l'app.
  • Passer → accès direct, sans code.

3

Le tableau de bord (vide au départ)

« Mes kits » — encore vide

Au tout début, aucun kit n'est enregistré. La carte « Mon parc » affiche 0 kit.

Le bouton + (en bas à droite) ajoute votre premier kit.

4

Ajouter un appartement (et choisir les lignes)

Fenêtre « Nouveau kit »

Renseignez le Nom du kit et le Numéro du kit (sa carte SIM), par ex. Loïc et 699173771.

💡 Nommez chaque kit d'après le logement (Appartement A, B…) pour vous y retrouver.

Nouveau Choisir le nombre de lignes

Avant de valider : 4, 7 ou Personnalisé (1 à 7, via − / +). Les lignes sont créées automatiquement (Ligne 1, Ligne 2…), à renommer ensuite (§7).

5

Votre parc complet

Trois appartements gérés

Après avoir répété l'opération, le bailleur gère Appartement A, B et C. La carte « Mon parc » indique 3 kits, 3 configurés.

Chaque carte montre le nom, le numéro du kit et un point de statut. Touchez-la pour ouvrir le logement.

6

Piloter les lignes d'un logement

L'onglet Lignes — le poste de commande

  • Trois onglets : Lignes, Conso, Config.
  • Compteur « 4/7 » : un kit gère jusqu'à 7 lignes, numérotées 1 à 7 (ordre des lignes dans le kit).
  • Actions rapides : Tout allumer / Tout éteindre.
  • Chaque ligne : son nom + deux boutons ON / OFF qui envoient la commande par SMS.

Des boutons d'action — l'état réel se lit dans l'historique

Les boutons ON / OFF sont volontairement neutres : appuyer ne fait qu'envoyer la commande au kit (r1on…r7on, où le chiffre est le numéro de ligne 1–7). L'app n'affiche aucun état « supposé » sur les boutons, pour ne pas induire en erreur.

Pour connaître l'état réellement confirmé, touchez l'icône horloge d'une ligne : son historique des accusés liste chaque confirmation (Allumée / Éteinte), le message exact renvoyé par le kit (ex. r1on) et l'heure.

⏳ Pendant l'envoi, le bouton pressé affiche brièvement « envoi en cours » et le contrôle est verrouillé pour éviter un double SMS.

🔑 L'historique fait foi. Un même kit peut être piloté par plusieurs téléphones autorisés ; les confirmations sont journalisées même application fermée ou depuis un autre onglet, et complètent l'historique automatiquement.
★

Le dialogue SMS, en images

Tout passe par SMS — voici le dialogue réel

Depuis la messagerie du téléphone, on voit le dialogue exact entre l'app et le kit :

  • Commander une ligne : l'app envoie r1on / r1off (le chiffre = le numéro de ligne 1–7).
  • Demander la consommation : l'app envoie cons ; le kit répond Consommation : 2.2 KWh.
  • Configurer le kit : l'app envoie n1:…:n2:…:en:…:ip:… ; le kit renvoie le même message en accusé de configuration.

📶 Aucune connexion Internet n'est requise : tout fonctionne par SMS, où que vous soyez.

7

Gérer une ligne (double-tap : renommer / supprimer)

Nouveau Édition par double-tap

Double-tapez une ligne : son nom devient modifiable, accompagné de deux boutons ronds épurés — ✓ vert (valider) et ✕ rouge (supprimer).

Saisissez le nouveau nom (Climatiseur, Chambre, Salon…) puis ✓. L'ampérage n'est plus demandé : une ligne se résume à son nom et son état.

Pour ajouter : « + Ajouter une ligne » (un nom suffit).

Des lignes parlantes

Nommez les lignes selon ce qu'elles alimentent : ici Climatiseur, Salon, Chambre… bien plus clair qu'un numéro.

8

Suivre la consommation

L'onglet Conso

« Actualiser la consommation » interroge le kit par SMS : il répond avec le relevé.

Les mesures s'accumulent ensuite dans l'Historique.

9

Configurer le kit

L'onglet Config — tout au même endroit

  • Informations du kit — nom et numéro du kit.
  • Compteur — impulsion par kWh et consommation initiale.
  • Numéros autorisés — qui peut commander par SMS.
  • Envoyer la configuration au boîtier (SMS).
  • Générer le QR Code de partage (§10).

Deux éditions via le crayon ✏️

  • Informations : corriger le nom et le numéro du kit.
  • Compteur : impulsion par kWh + consommation initiale (kWh).

Les numéros autorisés se saisissent directement dans deux champs de l'onglet Config (Numéro 1 / Numéro 2) — le kit n'en gère que deux — puis « Enregistrer ». Pratique pour habiliter un gardien ou un colocataire.

10

Partager la configuration par QR Code

Un atout majeur pour un bailleur : dupliquer ou transmettre la configuration d'un kit d'un téléphone à un autre, sans tout ressaisir.

Étape A Téléphone source

Config → « Générer le QR Code ». Le QR contient toute la configuration (numéro, lignes…). « Partager le QR Code » l'envoie aussi en image.

Étape B Téléphone destinataire

Touchez l'icône scanner (en haut à droite), autorisez la caméra, puis cadrez le QR du premier téléphone. Le kit est alors importé automatiquement.

💡 Cas d'usage bailleur. Transmettez par QR Code l'accès d'un logement précis au locataire ou au gardien — sans donner accès au reste du parc.   🛟 Sauvegarde : conservez une copie des QR pour restaurer un kit en cas de changement de téléphone. L'état des lignes se synchronise entre appareils autorisés.

Sécurité — code, empreinte et récupération

Nouveau Quatre réglages de sécurité

Code de sécurité : l'interrupteur active/désactive le code (activer → créer ; désactiver → saisir le code actuel). « Modifier le code d'accès » : ancien + nouveau + confirmation.

Déverrouillage par empreinte : ouvrez l'app avec votre empreinte digitale au lieu du code. Nécessite une empreinte enregistrée sur le téléphone ; sinon « Aucune empreinte enregistrée ». 🔑 Le code reste le secours.

Questions de récupération : deux questions secrètes qui permettent de retrouver l'accès via « Code oublié ? » si vous oubliez le code.

📦 Configuration groupée. Le bouton « Envoyer la configuration au kit » (§9) envoie tous les réglages dans un seul SMS, champs séparés par : — ex. n1:+237…:n2:+237…:en:…:ip:… — et un aperçu s'affiche pour validation avant l'envoi.   ⚠️ Ces mots-clés correspondent au langage du boîtier : ils ne se modifient pas côté application. Les numéros sont automatiquement mis au format +237.
🕘 Interprétation des réponses. Un message rNon / rNoff met à jour l'état de la ligne N sur tous les téléphones autorisés (même app fermée) et est horodaté dans l'historique des accusés de la ligne (§6.3), avec le texte brut reçu. Un message contenant « … kWh » est enregistré comme un relevé de consommation.

Récapitulatif — le parcours du bailleur

  1. Installer l'app ; créer un code seulement si souhaité — la page est skippable (§1–2, §12).
  2. Ajouter chaque appartement comme un kit, avec son numéro de kit et son nombre de lignes (§4–5).
  3. Nommer et piloter les lignes (numérotées 1–7) par double-tap, avec actions groupées et historique des accusés (§6–7).
  4. Suivre la consommation (§8).
  5. Configurer le kit : infos, compteur, numéros autorisés (§9).
  6. Partager / dupliquer la configuration par QR Code — l'état des lignes se synchronise (§10).
  7. Personnaliser et sécuriser l'application (§11–12), avec une FAQ (§13) et l'annexe SMS (§14).
